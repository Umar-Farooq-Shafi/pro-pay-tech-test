

## PropPay Test

### Laravel, React, Inertia and Tailwind CSS

### Download npm depends.
``npm install``

### Development watch.
``npm run hot``

### Production build.
``npm run build && npm run start``

### Config ``.env`` file.

### Run the migrations.
``php artisan migrate``

### Run the seeder
``php artisan db:seed``

### Start laravel project.
``php artisan serve``
